COMPILATION
1 - If not installed, install the ncurses.h library (sudo apt-get install libncurses5-dev)
2 - make

EXECUTION
1 - Server: ./Server
2 - Clients: ./ClientGUI <IP_SERVER>

COMMANDS CLIENT
1 - Change nickname: 		/n <name> 
2 - Enter in a chat room: 	/j <room_number> 
3 - Get back to the lobby: /l
4 - Leave the chat: 			/q
5 - To write a mesage: just type it
